📊 Calculadora de Juros de Dívida

Aplicação web simples que calcula os juros de uma dívida em atraso com base no valor informado e na quantidade de dias. O projeto foi desenvolvido com foco em prática de lógica, validação de dados e manipulação do DOM.

🚀 Funcionalidades
Inserção do valor da dívida
Inserção dos dias em atraso
Validação de dados (valores inválidos não são aceitos)
Cálculo automático de juros (1% ao dia)
Exibição do valor total atualizado
Feedback visual para erro e sucesso
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
💡 Regras de cálculo
Juros de 1% ao dia sobre o valor da dívida
Fórmula utilizada:
juros = valorDivida * 0.01 * diasAtraso
total = valorDivida + juros
⚠️ Validações

A aplicação impede:

Valores de dívida menores ou iguais a zero
Dias em atraso negativos
Campos vazios ou inválidos
📷 Interface
Layout centralizado
Design moderno com gradiente
Feedback visual:
🔴 Erro
🟢 Sucesso
